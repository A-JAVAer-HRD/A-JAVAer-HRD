1.CSS介绍

1.1什么是CSS

层叠样式表 (Cascading Style Sheets，缩写为 CSS）

CSS:表现（美化网页）

字体，颜色，编剧，高度，宽度，背景图片，网页定位，网页浮动…

1.2CSS发展史

CSS1.0

CSS2.0 DIV(块)+CSS,HTML与CSS结构分离的思想，网页变得简单，SEO

CSS2.1 浮动、定位

CSS3.0 圆角，阴影，动画…浏览器的兼容性~

1.3CSS快速入门

CSS 1.0:style CSS代码

css 3.0分离

代码分离，css通过链接link到html代码中。

css优势：

  1. 内容和变现分离
  2. 网页结构表现统一，可以实现复用
  3. 样式十分丰富
  4. 建议使用独立于html的css文件
  5. 利用SEO，更容易被搜索引擎收录

1.增加父级元素的高度

小结：

  1. 浮动元素后面增加空div 简单，代码尽量避免空的div
