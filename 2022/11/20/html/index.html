1、初识HTML

1.html

超文本标记语言（英语：HyperText Markup Language，简称：HTML）是一种用于创建网页的标准标记语言。

HTML是一种基础技术，常与CSS、JavaScript一起被众多网站用于设计网页、网页应用程序以及移动应用程序的用户界面[3]。

html:基础网页 html5

javaScript：动态渲染

5、列表及表单应用

1.from表单

form表单

action:表单提交的位置，可以是网站，也可以是一个请求处理的地址

method:post,get 提交方式

  • ​ get方式提交：我们可以在url中看到我们提交的信息，不安全，高效

  • ​ post:比较安全，不能传输大文件

2.文本框单选框

文本框选择标签：可以没有

  • value=”跳动的猪” 默认是初始值
  • maxlength=”8” 最大字符长度
  • size=”30” 文本框长度

单选框标签：至少有的三个值

  • input type=”radio” radio是单选框
  • value=”boy” 单选框的值
  • name=”sex” 表示组：只有值在一个组才可以单选

3.按钮多选框

多选框

  • input type=”checkbox” 多选框
  • value=”sleep” 值
  • name=”hobby” 分组

显示的是后边的文字，提交的值是value

按钮：

  • input type=”button” 普通按钮
  • input type=”image” 图像按钮 提交
  • input type=”submit” 提交按钮
  • input type=”reset” 重置

4.文本域、文件域、下拉框

文本域：

  • cols=”30”列
  • rows=”10” 行

文件域

  • input type=”file”
  • name=”files”

下拉框，列表框

  • select 列表
  • option 值
